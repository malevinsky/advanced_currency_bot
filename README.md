### Как высчитывается валюта
**Base** — базовая валюта, от которой высчитываются значения остальных. 

Например, мы знаем, что: 
1 USD = 0.87 EUR, 
1 USD = 0.73 GBP, 
Чтобы узнать, сколько EUR равняется GBP?

Формула: 
```
a / b = c
```

```
EURGBP = (USDGBP / USDEUR) = (0.73 / 0.87) = 0.84
```
